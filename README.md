# 📝 ToDo App

A simple and interactive ToDo application built with **ReactJS**. It allows users to add, display, and manage daily tasks efficiently.

## 🚀 Features

- Add new tasks
- View added tasks
- Clean and responsive UI
- Built with React and Bootstrap

## 🛠️ Technologies Used

- ReactJS
- JavaScript (ES6+)
- CSS
- Bootstrap 5
